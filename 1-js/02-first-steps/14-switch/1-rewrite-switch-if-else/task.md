# در کد زیر "switch" را به "if" تبدیل کنید

کد زیر را با استفاده از `if..else` بازنویسی کنید

```js
switch (browser) {
  case "Edge":
    alert("You've got the Edge!");
    break;

  case "Chrome":
  case "Firefox":
  case "Safari":
  case "Opera":
    alert("Okay we support these browsers too");
    break;

  default:
    alert("We hope that this page looks ok!");
}
```
