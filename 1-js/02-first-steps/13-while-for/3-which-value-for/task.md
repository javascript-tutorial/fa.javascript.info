# کدام مقدارها توسط حلقه "for" نمایش داده می شوند؟

برای هر حلقه یادداشت کنید که چه مقداری را نمایش خواهد داد. سپس با جواب مقایسه کنید.

هر دو حلقه مقدارهای مشابه را `alert` می کنند یا نه؟

1. شکل پسوند:

    ```js
    for (let i = 0; i < 5; i++) alert( i );
    ```
2. شکل پیشوند:

    ```js
    for (let i = 0; i < 5; ++i) alert( i );
    ```
