# دستور 'if..else' را به '?' بازنویسی کنید

با استفاده از چند عملگر سه‌گانه `'?'` دستور `if..else` را بازنویسی کنید.

برای خوانایی، پیشنهاد می‌شود که کد را در چند خط بنویسید.

```js
let message;

if (login == 'کارمند') {
  message = 'سلام';
} else if (login == 'مدیر') {
  message = 'درود';
} else if (login == '') {
  message = 'وارد سیستم نشدید';
} else {
  message = '';
}
```
