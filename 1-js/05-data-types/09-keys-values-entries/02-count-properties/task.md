# ویژگی‌ها را بشمارید

یک تابع `count(obj)` بنویسید که تعداد ویژگی‌های درون شیء را برمی‌گرداند:

```js
let user = {
  name: 'John',
  age: 30
};

alert( count(user) ); // 2
```

سعی کنید تا جایی که می‌توانید کد را کوتاه بنویسید.

پی‌نوشت: ویژگی‌های سمبلی را نادیده بگیرید، فقط ویژگی‌های «معمولی» را بشمارید.
