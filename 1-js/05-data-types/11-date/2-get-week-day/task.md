# روز هفته را نشان دهید

یک تابع `getWeekDay(date)` بنویسید که روز هفته را به این شکل نشان می‌دهد: 'MO'، 'TU'، 'WE'، 'TH'، 'FR'، 'SA'، 'SU'.

برای مثال:

```js no-beautify
let date = new Date(2012, 0, 3);  // 3 Jan 2012
alert( getWeekDay(date) );        // را نشان دهد "TU" باید
```
